# Helpers

## GIT
add new commit
```
git add -A && git commit -m "Commit message"
```


push
```
git push
```


git push tag
```
git push --tags
```


Delete release from repo:
```
git push --delete origin v1.0.2
```


## NPM

Publish package
```
npm publish
```

Bump new version
```
npm version patch
```
